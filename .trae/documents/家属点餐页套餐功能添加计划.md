# 家属点餐页套餐功能添加计划

## 概述

在家属角色点餐页面 (`user-order/index.vue`) 中添加套餐（Setmeal）支持。当前页面只有菜品（分类→菜品）的浏览和购物车操作，需增加套餐的浏览与加入购物车功能。后端接口已就绪，主要改动在前端。

## 后端接口现状（无需改动）

| 接口 | 方法 | 说明 |
|------|------|------|
| `GET /user/setmeal/list?categoryId=` | GET | 按分类获取套餐列表，返回 `List<Setmeal>`，不传 categoryId 返回全部 |
| `GET /user/setmeal/dish/{id}` | GET | 获取套餐详情（含套餐内菜品列表），返回 `SetmealVO` |

- `ShoppingCart` 实体已有 `setmealId` 字段
- `ShoppingCartPayload` 已支持 `setmealId` 参数
- `addShoppingCart({ elderId, setmealId })` / `subShoppingCart({ elderId, setmealId })` 可直接使用
- `ShoppingCartItem` 接口已保留 `setmealId` 字段

## 前端改造点

### 改造文件清单

|------|----------|------|

### 步骤 3：`src/utils/familyCart.ts` — 添加套餐购物车数量映射

在 `buildDishQuantityMap` 函数下方，新增 `buildSetmealQuantityMap` 函数：

```ts
export type SetmealQuantityMap = Record<number, number>

export function buildSetmealQuantityMap(items: ShoppingCartItem[]): SetmealQuantityMap {
  return items.reduce((quantityMap, item) => {
    if (item.setmealId === undefined || item.setmealId === null) {
      return quantityMap
    }
    const key = Number(item.setmealId)
    quantityMap[key] = (quantityMap[key] || 0) + Math.max(0, Number(item.number || 0))
    return quantityMap
  }, {} as SetmealQuantityMap)
}
```

**验证**：函数能正确聚合套餐购物车项的数量。

##### 修改 `loadDishesForDiningPoint` 方法

在选择助餐点后同时加载菜品和套餐（或懒加载套餐）：

```ts
private async loadDishesForDiningPoint(diningPointId: number, preserveCurrentCategory = true) {
  this.dishLoading = true
  try {
    const response = await getFamilyDishList({ diningPointId })
    const payload = this.extractPayload<Dish[]>(response)
    this.applyDishList(Array.isArray(payload) ? payload : [], preserveCurrentCategory)
  } catch (error) {
    this.resetDishData()
    this.$message.error(this.resolveErrorMessage(error, '菜品加载失败，请稍后重试'))
  } finally {
    this.dishLoading = false
  }
  // 同时预加载套餐
  this.loadSetmealsForDiningPoint(diningPointId)
}
```

**`selectSetmealCategory(categoryId)`** — 点击套餐分类项：
```ts
private selectSetmealCategory(categoryId: number) {
  if (this.currentSetmealCategory === categoryId) return
  this.currentSetmealCategory = categoryId
  this.currentSetmeals = this.setmealCategoryMap[categoryId] || []
}
```

**`getSetmealCartQuantity(setmealId)`** — 获取套餐在购物车的数量：
```ts
private getSetmealCartQuantity(setmealId: number) {

```

**`increaseSetmealQuantity(setmeal)` / `decreaseSetmealQuantity(setmeal)`** — 套餐加入/减少购物车：
参照 `increaseQuantity(dish)` / `decreaseQuantity(dish)` 的代码结构，使用 `setmealId` 代替 `dishId`：
```ts
private async increaseSetmealQuantity(setmeal: Setmeal) {
  if (!this.ensureDishCanBeAdded()) return
  const elderId = this.requireCartActionElderId()
  if (!elderId) return
  this.cartSyncing = true
  try {
    await addShoppingCart({ elderId, setmealId: setmeal.id })
    await this.refreshCartState()
  } catch (error) {
    this.$message.error(this.resolveErrorMessage(error, '加入购物车失败，请稍后重试'))
  } finally {
    this.cartSyncing = false
  }
}

private async decreaseSetmealQuantity(setmeal: Setmeal) {
  const elderId = this.requireCartActionElderId()
  if (!elderId) return
  this.cartSyncing = true
  try {
    await subShoppingCart({ elderId, setmealId: setmeal.id })
    await this.refreshCartState()
  } catch (error) {
    this.$message.error(this.resolveErrorMessage(error, '减少套餐失败，请稍后重试'))
  } finally {
    this.cartSyncing = false
  }
}
```

**`resetSetmealData()`** — 重置套餐数据：
```ts
private resetSetmealData() {
  this.allSetmeals = []
  this.visibleSetmealCategories = []
  this.setmealCategoryMap = {}
  this.currentSetmealCategory = null
  this.currentSetmeals = []
}
```

## 数据流示意

```
用户点击「套餐」标签
  → switchOrderingMode('setmeal')
  → 渲染 visibleSetmealCategories（左侧分类列表）
  → 默认选中第一个套餐分类 → selectSetmealCategory(id)
  → 渲染 currentSetmeals（右侧套餐网格，使用 SetmealCard）
  
用户点击套餐的「+」按钮
  → increaseSetmealQuantity(setmeal)
  → addShoppingCart({ elderId, setmealId })
  → refreshCartState()
  → cartBar / cartDrawer 自动更新
```
